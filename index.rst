Welcome to FinAI Projects! - A curated repository of AI and ML projects and resources in Finance
===========================================================

Welcome to the Documentation for `FinAILab Projects <http://finailab.com/>`_. This documentation contains the curated list of different python based projects and github repository in Machine Learning in Finance.

Key highlights of Fin AI Projects
--------------------------------------
- Curated projects across different AI and Machine Learning type (i.e. Supervised Learning, Unsupervised Learning, Reinforcement Learning and NLP)
- Curated projects across following applications in Finance:

    - Trading Strategies and Algorithmic Trading
    - Portfolio Management
    - Derivatives Pricing and Hedging
    - Asset Price Prediction
    - Fraud Detection
    - Loan Default probability prediction
    - Chatbot and automation

- Plug and play github repositories - just plug in the data and build models.
